# IP Info Analyzer

Консольное приложение на C#, которое:
- Загружает список IP-адресов из файла
- Получает информацию по каждому IP через API ipinfo.io
- Определяет, какие страны представлены и сколько IP относится к каждой
- Выводит города для страны с максимальным количеством IP

## Используемые технологии
- C#
- HttpClient
- Newtonsoft.Json

## Запуск
1. Поместить файл ips.txt в корень проекта
2. Запустить приложение
